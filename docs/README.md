# Docs Overview

`docs/` je teď rozdělené podle typu informace, ne podle historického původu souborů.

## Struktura

```text
docs/
  architecture/   current architecture notes and architecture changelog
  specs/          product / implementation specs
  plans/          implementation plans
  reference/      external or archived reference materials
  assets/         screenshots and other non-code supporting assets
  superpowers/    agent-specific plans/specs kept as-is
```
