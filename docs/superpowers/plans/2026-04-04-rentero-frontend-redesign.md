# Rentero Frontend Redesign — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the current mixed Tailwind/glassmorphism frontend with a unified "Dark SaaS Pro" design system across all 10 templates.

**Architecture:** Single CSS stylesheet embedded in `base.html` using CSS custom properties. All templates extend `base.html` and use its component classes. Tailwind CDN removed from all files. Minor additions to `web.py` dashboard route to supply portfolio-level KPI aggregates. No new files created.

**Tech Stack:** FastAPI + Jinja2 + HTMX (unchanged). Inter font via Google Fonts. Inline SVG sparklines generated by Jinja2. CSS `@keyframes` for all animations.

**Verification method for all tasks:** Run `python run_web.py` from the project root, open `http://localhost:8000` in a browser, verify the screen visually. No automated tests exist or are needed for pure HTML/CSS work.
